# AI Resume Analyzer & Career Recommendation System

This project analyzes resumes using Python and NLP.  
It extracts skills, calculates ATS score, detects missing skills, and recommends career paths.

## Features
- Resume Upload
- Skill Extraction
- ATS Score
- Career Recommendation
- Graph Visualization
- Login System
- Downloadable Report

## Technologies Used
Python, Streamlit, NLP, Machine Learning

## How to Run
pip install streamlit  
streamlit run app.py
